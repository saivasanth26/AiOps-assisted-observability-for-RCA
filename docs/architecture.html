Root Cause Analysis
Deterministic rule engine
1
Cascade Failure
Errors detected across multiple services in the trace. Identifies first failing service in call chain.
2
Resource Pressure
CPU usage exceeds 80% threshold. Service under sustained resource pressure.
3
Traffic Spike
RPS exceeds 2x baseline. Load pattern causing saturation.
4
Downstream Dependency
Downstream service span duration exceeds target service internal duration. Bottleneck is external.
5
Internal Slowness
Default fallback. No external cause identified. Profile the service for internal bottlenecks.
